# proyecto cv interactivo - daniela torres

este proyecto implementa un curriculum vitae web utilizando tecnologías frontend fundamentales, manteniendo una estricta separación de responsabilidades: estructura (html), estilo (css) y comportamiento (js).

## estructura del proyecto
- **index.html**: contiene únicamente la estructura semántica y clases de componentes esenciales. se ha evitado el uso de clases utilitarias de estilo en el html.
- **style.css**: contiene todas las reglas visuales, márgenes, colores y alineaciones. incluye variables css para facilitar el modo oscuro.
- **script.js**: maneja la lógica de interacción utilizando jquery y javascript.

## características
1. **diseño responsivo**: adaptable a móviles y escritorio.
2. **modo oscuro**: funcionalidad para cambiar el tema visual.
3. **validación**: feedback visual en el campo de email.
4. **animaciones**: aparición suave de secciones al hacer scroll.

## tecnologías
- html5
- css3
- bootstrap 5 (solo componentes y grid)
- javascript / jquery

## cómo ver el proyecto
abre el archivo `index.html` en tu navegador. requiere conexión a internet para cargar bootstrap y jquery desde cdn.
